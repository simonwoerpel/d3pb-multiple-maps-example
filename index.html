d3-playbooks-maps example

correctiv superbugs maps

See original publication at https://correctiv.org/superbugs

This example demonstrates how to use d3-playbooks-maps to render multiple responsive static maps efficiently. See README for details.

change colors

This shows how to manipulate the rendered maps once they are created, for example change colors afterwards. It's also explained in the README how to do that.

Click on one of the buttons to update the colorScheme. The schemes are all from d3-scale-chromatic. I personally recommend the RANDOM-button.
